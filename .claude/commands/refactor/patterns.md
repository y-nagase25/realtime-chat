# Common Refactoring Patterns

## 1. Extract Custom Hook
Convert stateful logic in components to custom hooks.

## 2. Extract Server Action
Move data mutations to Server Actions.

## 3. Simplify Conditionals
Replace nested if-else with early returns or switch statements.

## 4. Extract Configuration
Move magic numbers and strings to constants.

## 5. Introduce Type Guards
Add type guards for runtime type checking.

## 6. Extract Utility Functions
Move reusable logic to lib/utils.
